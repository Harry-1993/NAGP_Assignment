# OrdersService

## Provide a REST endpoint giving details of all orders of a user 

**End point:** GET /orders/1

```json
{
  "orders": [
    {
      "orderId": 1,
      "orderAmount": 250,
      "orderDate": "14-Apr-2020"
    },
   {
      "orderId": 2,
      "orderAmount": 450,
      "orderDate": "15-Apr-2020"
    }
  ]
}
```
